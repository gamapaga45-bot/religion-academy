\# Сравнительное Религиоведение (Академический курс)



Это одностраничное веб-приложение (React SPA), предназначенное для академического изучения и параллельного сравнения священных текстов авраамических традиций (Библии и Корана).



\## Архитектура

Проект построен по принципу бессерверной архитектуры (Serverless + CDN). 

Сам репозиторий не хранит тяжелых текстовых баз данных. Вместо этого приложение «на лету» обращается к открытым API и публичным репозиториям для загрузки необходимых текстов.



\* \*\*Frontend:\*\* React 18, Tailwind CSS, Babel (Standalone)

\* \*\*Библия (API):\*\* Синодальный перевод (подгружается через GitHub Raw)

\* \*\*Коран (API):\*\* Перевод Э. Кулиева (подгружается через jsDelivr CDN / Alquran API)



\## Как запустить локально

1\. Клонируйте репозиторий: `git clone <ваш-url>`

2\. Откройте файл `index.html` в любом современном браузере. (Для избежания ошибок CORS рекомендуется запускать через локальный сервер, например, расширение \*Live Server\* в VS Code).



\## Деплой (GitHub Pages)

Проект является 100% статичным. 

1\. Перейдите в настройки репозитория (Settings) -> Pages.

2\. В разделе "Build and deployment" выберите ветку `main`.

3\. Сохраните. Сайт будет доступен по ссылке через 1-2 минуты.
